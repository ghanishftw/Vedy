# Product

## Register

product

## Platform

web

## Users
Creators and casual users who want to quickly download and edit/trim YouTube clips. They need a frictionless, fast, in-browser interface with zero sign-ups.

## Product Purpose
Vedy provides an instant, private, client-side tool to trim and download YouTube videos (quality selections up to 1080p and audio-only MP3) directly in the browser using FFmpeg.wasm and range-aware server proxies.

## Positioning
The Next Gen Youtube Downloader: fast, client-side, zero signup video snips.

## Brand Personality
Sleek, creator-centric, technically premium, modern dark tool.

## Anti-references
- Clunky ad-ridden YouTube converters.
- Slow SaaS tools gating downloads behind authentication walls.
- Complex desktop video editing suites.

## Design Principles
- **Frictionless speed:** Zero steps between loading the page, pasting a link, and exporting.
- **Real-time feedback:** WYSIWYG filter adjustments and frame-exact timeline controls.
- **Creator focus:** Visual style that feels premium, like a professional developer/creator utility, avoiding generic consumer software tropes.

## Accessibility & Inclusion
- Support keyboard controls for video play/pause and timeline scrubbing.
- Respect reduced motion media queries for transitions and loading state animations.
- Minimum WCAG AA contrast for text overlays.
